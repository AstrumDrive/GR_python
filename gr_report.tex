\subsection{Tetrad Construction}
The metric has an \textbf{ADM shift vector} with a diagonal spatial block ($\gamma_{ij} = 0$ for $i \ne j$). The tetrad was built via the ADM decomposition: lapse $N$, shift $\beta^i$, and spatial triad $E^{\hat{a}}{}_i = \sqrt{\gamma_{ii}}\,\delta^{\hat{a}}_i$ (no sum).
\subsection{ADM Lapse and Shift}
The ADM decomposition gives lapse $N$ and shift $\beta^i$:
\begin{dmath*}
  N = 1
\end{dmath*}
\begin{dmath*}
  \beta^{r} = \beta{\left(r \right)}
\end{dmath*}
\begin{dmath*}
  \beta^{\theta} = 0
\end{dmath*}
\begin{dmath*}
  \beta^{\varphi} = 0
\end{dmath*}
\subsection{Spatial Triad $E^{\hat{a}}{}_i$}
The spatial triad satisfies $\gamma_{ij} = \delta_{\hat{a}\hat{b}}\,E^{\hat{a}}{}_i\,E^{\hat{b}}{}_j$.
